    • Features

    • Search with Google Maps API

      Allows user to search for gym locations near them using the Google Maps API.

    • Viewing of reviews with Google Places API

      Allows users to see how good or bad the selected gym location is, and that way the user can decide if he/she wants to go that gym. Using Google Places API.

    About

    This project was inspired by the frustration of visiting gyms only to be disappointed by what was found. With GainzHunt, anyone can get into the fitness lifestyle in a location that best suits their needs, saving time and effort.
